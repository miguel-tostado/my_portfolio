About Me

Hi! My name is Miguel Tostado and I'm a freelance web developer located in Maine who specializes in web design and front-end development. My goal with each project I design and build is to create a website that appeals to the user base while maintaining a high level of performance.

As a designer, I aim for a clean, modern style that's very intuitive and dynamic with a focus on converting users to customers.

As a developer, I want my websites to be fast, responsive, and reliable.

Having been in the US Navy for 6 years, I'm no stranger to hard work and dedication. When I start I project I will make it a priority and ensure the final product is something I can be proud of and happily attach my name to.
